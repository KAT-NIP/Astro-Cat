# Astro-Cat
2021-1 MME 3D Final Project Astro-Cat</br></br>

  > ### 게임 설명
  주인공, 고양이 냥냥이가 우연히 우주로 떨어져서 주인에게 돌아가기 위해 다양한 행성에서 미니게임을 진행하며 보석을 세 개 모아서, 열쇠와 교환해 주인에게 다시 돌아오는 이야기를 담은 힐링 게임
  
  > ### 게임 장르
  퍼즐, 어드벤처, TPS, 미니게임
